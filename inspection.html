Inspecting your technical documents with IBM Deep Search

IBM Deep Search provides a simple utility to view the processing results of a single document. Use this utility to assess the way in which IBM Deep Search parses and understands your proprietary documents.

+ Expand All

- Close All
Start Tutorial
Step 1: Click on the top menu bar item labeled ``Inspection''.

Step 2: On the next page you can drag and drop a single file to the page, or you can click on the link in the center of the page to open a file selection dialog box.

Step 3: If using the file selection dialog box, select the file for analysis.

Step 4: When the document has been completely parsed it will be presented to you as shown below.

The view is divided into two halves. On the right side you will see the original PDF file with blocks overlayed on the text. The blocks define specific parts of the document identified by IBM Deep Search. On the left you will see the text extracted from each block.

Step 5: Click on any of the blocks in the PDF document to view the corresponding extracted text on the left side of the screen.

Step 6: You may also download and view the JSON object text file that was generated for the document by clicking on the download icon in the upper right of the document viewer.
